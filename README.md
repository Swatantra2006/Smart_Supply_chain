# Resilient Logistics – Smart Supply Chain Dashboard

A real-time supply chain risk monitoring and disruption management platform.

## Deploy to Vercel

1. Upload this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) → Import Project → Select your repo
3. Vercel will auto-detect settings — just click **Deploy**

## Project Structure

```
resilient-logistics/
├── index.html       ← Main dashboard (single-file app)
├── package.json     ← Required by Vercel
├── vercel.json      ← Vercel deployment config
└── README.md
```

## Features
- Live shipment tracking (38 shipments)
- AI risk classification (High / Medium / Low)
- Disruption simulation & rerouting
- Analytics dashboard with Chart.js
